# Golden Elephant Events

Beautifully designed Next.js app for booking event decor services.

## 🚀 Quick Start

```bash
npm install
npm run dev
```

## 🌐 Deploy to Vercel

1. Push to GitHub
2. Visit https://vercel.com/import
3. Select your repo and deploy
